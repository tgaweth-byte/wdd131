CSS Layout Review

Understanding basic layout techniques

1. Float Layout

Float moves elements to the left or right, and other content wraps around it.

This text wraps around the floated image. Float was commonly used for layouts before Flexbox and Grid existed. Notice how the text flows around the image on the right side.

2. Flexbox Layout

Flexbox arranges items in a row or column and gives you control over spacing and alignment.
